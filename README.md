> List extraction from json or csv input


## Install

```sh
$ npm install --save lsty
```


## Usage

```js
var lsty = require('lsty');

lsty('Rainbow');
```

```sh
$ npm install --global lsty
$ lsty --help
```

```sh
# creates a browser.js
$ npm run browser
```
